7. Features Not Included in the Basic (Starter) Plan

The following features are not available on the Starter plan and require an upgrade to Pro, Premium, or Enterprise:

  • Hybrid search (keyword + semantic)
  • Unlimited version history
  • Custom branding
  • Custom reports, saved views, and scheduled report delivery
  • ML-powered anomaly detection
  • Natural language query
  • Compliance Suite, SOC 2 evidence automation, and GDPR DSAR automation
  • HIPAA-compliant environment
  • Extended audit log retention (beyond 90 days)
  • Webhooks
  • Slack and Salesforce integrations
  • SSO (Okta, Azure AD, SAML 2.0), custom OAuth apps, and SCIM provisioning
  • In-app chat support, priority response SLA, and dedicated support channels
  • EU-West data residency
  • Uptime SLA with service credits
  • Sandbox environments
  • BYOK encryption

Starter plan customers have access to the REST API, email support, Zapier integration, basic usage dashboard, 5 pre-built report templates, 90-day audit log, and up to 1 GB/month data export.
